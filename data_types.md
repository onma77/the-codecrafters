# data types
data types tells the computer how to interpret and use a piece of data.

# type of data:
1. int: can only take numbers e.g
* Signed integers - can store both positive and negative values
* Unsigned integers - can only store non-negative values


2. string: takes in text, String values must be surrounded by double quotes

3. bool can only take the values true or false.
4. float: takes numbers like decimal.
